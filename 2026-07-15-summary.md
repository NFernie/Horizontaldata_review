## 1. Project in one paragraph

Python pipeline processes **23 horizontal McKinlay-member wells** (cuttings, LAS, mudlogs, trajectory) into JSON under `site/public/data/`, plus interpretation Markdown in `output/`. A React/Vite/Tailwind static site visualises **intra-well stats**, **inter-well comparison** (Jaccard, hierarchical clustering, KS), **executive summary** (Jena focus), and **water-risk exploration**. Primary stakeholder objective: assess **water-risk zones** in **JENA 31** and **JENA 31DW1** using analog wells and statistical methods defined in `New Statistical Methods.md` / `OpusPlanR1.md`.
