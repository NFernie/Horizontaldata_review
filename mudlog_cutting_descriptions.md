# Mudlogging Cuttings Descriptions

Reference glossary for abbreviated cuttings descriptions used in McKinlay mudlog and sample interpretation text.  
**Purpose:** Cursor / parser reference only — not for updating sample spreadsheet files.

---

## CUTTING DESCRIPTIONS

### SANDSTONE (SST)

#### Colour

gry (grey), brn (brown), grn (green), tn (tan), blk (black), blu (blue), yel (yellow), orng (orange), wh (white), off wh (off white), clr (clear) or col (colourless), trnsl (translucent), frstd (frosted), mlky (milky), trnsp (transparent), opq (opaque), dk (dark), lt (light), pl (pale), dsky (dusky), mott (mottled), bff (buff), rd (red), pnk (pink), crm (cream).

#### Grain Size

vf (very fine), f (fine), med (medium), crs (coarse), v crs (very coarse), gran (granules), grs (grains).

Also use **pred** (predominantly), **dom** (dominantly), **to occ** (occasionally) or **mnr** (minor) as prefixes where applicable in the case of assorted grain size in the one sample.

#### Sorting (srt)

| Abbrev | Meaning |
|--------|---------|
| v pr | very poor |
| pr | poor |
| mod | moderate |
| mod wl | moderately well |
| wl | well |
| v wl | very well |

#### Angularity (ang)

ang (angular), sbang (subangular), sbrnd (subrounded), rnd (rounded), v wl rnd (very well rounded).

#### Cement (cmt)

| Type | Description |
|------|-------------|
| calc cmt | calcareous — fizz with HCl |
| sil cmt | siliceous — no fizz with HCl |
| arg cmt | argillaceous |
| sid cmt | sideritic — red rust |

Plus a modifier: **strg** (strong), **mod** (moderate), **wk** (weak) or **tr** (trace).

Write as **colour-modifier-description**.

#### Matrix (mtx) — between cement & grain (other stuff)

arg mtx (argillaceous), sil mtx (siliceous), slty mtx (silty), aren mtx (sandy), kaol mtx (kaolinitic — white powder; may be rock flour so be sure!).

Write as **quantity-colour-description**.

#### Minor or Trace Minerals

Use **i/p** (in parts), **spks** (specks), **tr** (trace), **occ** (occasionally), **tr-com** (trace to common), **rr** (rare 1–5%), **mnr** (minor 5–10%), **mod abdt** (moderately abundant 10–15%), **abdt** (abundant 15–20%).

| Mineral | Notes |
|---------|-------|
| pyr (pyrite) | Gold coloured; can be nod (nodular) or dissem (disseminated) |
| glauc (glauconite) | Green; associated with marine deposits |
| sid (siderite) | brn/orng, dirty rd/brn (FeCO₃); reacts with HCl; com with deltaic deposits |
| feld (feldspar) | rd/brn, gry, wh; looks like clay but very hard |
| carb (carbonaceous) | use spks, frags (fragments), flks (flakes or flecks), incl (inclusions), etc. |
| calcite | com hard grd to LMST (may fluoresce orange) |
| mic (mica) | also micmic (micromicaceous) |
| qtz (quartz) | generally as large angular frags or aggs (aggregates) and qtz ovgths (quartz overgrowths — large chunks of clear quartz bigger than rest) |
| liths (lithics) | unidentified rock/mineral fragments |
| foss (fossiliferous / fossil fragments) | usually name the fossil or dominant fossil e.g. Tr inoc (inoceramus) |
| chlor (chlorite) | generally dark green |

Write as **quantity-mineral**.

#### Hardness

lse (loose), uncons (unconsolidated), fri (friable), mod hd (moderately hard), hd (hard), v hd (very hard).

Avoid using **sft** (soft) or **frm** (firm) in SST descriptions.

#### Porosity (por)

nil, ti (tight), v pr (very poor), pr (poor), fr (fair), gd (good).

- **vis por** (visible porosity) — visual porosity for aggregates.
- **inf por** (inferred porosity) — inferred porosity for loose grains. When both aggregates and loose grains occur in the same sample, make sure both porosities are described.

#### Other common terms

- **qtz ovgths** (quartz overgrowths) — large chunks of clear quartz bigger than rest; can see flat surfaces.
- **bit sep lse qtz** (bit separated loose quartz) — loose sand grains not cemented.

#### Fluorescence (fluor)

1. Interval e.g. 5120'–5230'
2. Percentage of fluorescent grains
3. Brightness: dim, v dll (very dull), dll (dull), pl (pale), mod bri (moderately bright), bri (bright)
4. Colour e.g. pl yel/grn
5. Distribution: ppt (pinpoint), sptd (spotted), ptchy (patchy), evn (even)
6. Cut description:
   - **Activity:** v slw (very slow), slw (slow), mod rapid, fst (fast), inst (instant)
   - **Type:** bldng (bleeding), strmng (streaming), blmng (blooming), etc.
   - **Direct cut or c/c** (crush cut). If direct cut, don't mention "direct" in description. E.g. slw strmng cut.
   - **Colour.** E.g. pl yel, yel/wh. E.g. vslw, pl yel c/c.
7. **Residue:** ptchy (patchy), thn (thin), thk (thick), flm res (film residue), r/r (ring residue)

**Note:** Difference between oil fluorescence and mineral fluorescence (e.g. Limestone/Dolomite) is that mineral fluorescence doesn't move when solvent is added.

### SILTSTONE (SLTST) & CLAYSTONE (CLYST)

#### Colour

Same as SST.

#### Major features or types

arg (argillaceous — clayey; easily broken down to mud), aren (arenaceous — sandy), calc (calcareous — HCl fizz), carb (carbonaceous — black specks like coal), sil (siliceous), glauc (glauconitic — green specks).

#### Minor features or trace minerals

a/a. micmic (micromicaceous), liths (lithics — unidentified), pyr (pyrite), carb spks (carbonaceous specks), etc.

#### Hardness

disp (dispersive), v sft (very soft), sft (soft), frm (firm), mod hd (moderately hard), hd (hard), v hd (very hard).

#### Fissility or structure

fiss (fissile — platey), sbfiss (subfissile), sbblky (subblocky), blky (blocky — blocks), amorph (amorphous — very soft like clay; breaks down easily).

---

### LIMESTONE (LMST) & DOLOMITE (DOL)

#### Colour

Same as SST.

#### Type

micr (micritic — very fine), spar (sparitic — visible grains), lut (lutitic — no visible grains), rud (ruditic — pebble size).

#### Major features

arg (argillaceous — clayey), aren (arenaceous), pell (pelloidal — pellet like), ool (oolitic — ball like), foss (fossiliferous).

#### Minor features

a/a. cryptxln (cryptocrystalline), xln (crystalline), micrxln (microcrystalline).

#### Hardness

Same as SST.

Use porosity & fluorescence if the Limestone or Dolomite is a potential reservoir or target. Limestone and Dolomite will show fluorescence like white paper. Difference between oil fluorescence and mineral fluorescence (e.g. Limestone/Dolomite) is that mineral fluorescence doesn't move when solvent is added.

---

### COAL

#### Colour

Same as SST.

#### Lustre

dll (dull), ea (earthy), sbvit (subvitreous), vit (vitreous — glassy, shiny), wxy (waxy).

#### Other features

arg (argillaceous — clayey), slty (silty), bit (bituminous), lig (lignitic), striat (striations/striated), plnt rem (plant remains).

#### Hardness

v sft (very soft), sft (soft), brit (brittle), frm (firm), mod hd (moderately hard), hd (hard), v hd (very hard).

#### Fissility

fiss (fissile), sbfiss (subfissile), sbblky (subblocky), blky (blocky), amorph (amorphous).

#### Fracture

hckly (hackly), unevn (uneven), evn (even), ang (angular), conch (conchoidal), sbconch (subconchoidal).
